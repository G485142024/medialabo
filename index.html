「Webコンテンツ作成」の成果

本テーマについて

先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することが目的の講義です。

到達目標は次の4つです。

  1. 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できる.
  2. コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できる.
  3. モーション解析・編集ができる.
  4. HTML等を用いたWebページ作成ができる。

プロジェクト課題について

  • グルメ情報を検索できるページです。
  • 検索キーにコードを入れると店の情報が表示されるページです。
  • 店の情報はAjax通信により送信したリクエストで取得しています。
  • フォントや色選びがどうやってもダサくなってしまうのでいっそシンプルなデザインにまとめました。
  • 隙間なく文が並んでいて見づらかったため店ごとに四角で囲んで見やすいようにする処理を加えました。
  • サブジャンルがない店があるためサブジャンルの有無を判定してない場合スキップする処理を加えました。

    数当てゲーム

  • 回答回数kaisuにhantei()の度に1を加える処理を加えました。
  • 「kaisu回目の予想:yoso」を出力する処理を加えました。
  • 条件分岐してコンソールにメッセージを出力するような処理を加えました。
  • kaisuが4以上のときに「答えは ◯ でした．すでにゲームは終わっています」を出力する処理を加えました。

    以下課題4-1の実施内容

  • kazuate.html の最初の p 要素の中のテキストを削除し、ラベル、テキストボックス、回答ボタンを配置しました。
  • kazuate.js においてhantei() の呼び出し文 4つを全て削除、回答ボタンのクリックイベントハンドラとして hantei を登録をしました。
  • hantei() の定義において変数 kaisu の値を要素 span#kaisu のテキストに設定、yoso = 4; の 4 の代わりにテキストボックスの値を指定、yoso の値を要素 span#answer のテキストに設定する、メッセージの出力先を，コンソールから要素 p#result 要素のテキストに切り替えるをしました。

    各授業回の学習事項

    第1回 WWW と HTML

  • htmlは様々な要素で作られている。
  • htmlの文書は決まった要素で構成されている。
  • htmlのテキストの要素は<>で区切ることで使うことができる。
  • htmiで特殊文字を使う時は特有の記述をする必要がある。
    •

    演習の成果:課題1-4:HTMLの様々な要素

    第2回 CSS による Web ページの装飾

  • CSSによりページを装飾することができる。
  • CSSを使うことで文字の色やフォントを変えることができる。
  • 他に背景色を追加したりグラデーションしたりすることもできる。
  • 枠組みや表などの形式を自由に変えることもできる。
    •

    演習の成果:課題2-3

    第3回 JavaScript の基本

  • JavaScriptによってWebページ上で複雑な機能を実現することができる。
  • JavaとJavaScriptは名前が似ているが別の仕組みを持つ言語である。
  • JavaScript では「変数の型」という考え方をしない。
  • Firefox のウェブ開発ツールを使ってJavaScriptプログラムのデバッグをすることができる。
    •

    演習の成果:課題3-6

    第4回 DOM と JavaScript

  • ボタンを押したらページが変化する処理のプログラムがある。
  • ブラウザによって表示するユーザインタフェースに違いがある。
  • HTML文書の中の要素をJavaScriptプログラムで検索したり、要素が持つテキストなどの値を取得したり変更する機能はDOMにより実現される。

    演習の成果:課題4-10

    第5回 イベント処理

  • DOMはWebコンテンツを表現するデータ構造である。

    工夫点，苦心点，強調点，感想
